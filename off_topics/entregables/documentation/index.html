Introducción

Github Pages es un servicio de alojamiento web gratuito que permite a los usuarios publicar sitios web estáticos generados con tecnologías web como HTML, CSS y JavaScript.

Este manual proporciona una guía paso a paso para crear un sitio web utilizando Github Pages.

  • configuración de un repositorio de Github
  • creación de un sitio web estático
  • publicación del sitio web en Github Pages
  • obtención de la URL final del sitio web para compartir con otros
  • supervisión del proceso de implementación y solución de problemas
Primer commit

Para comenzar, cree un nuevo repositorio en Github y clone el repositorio en su máquina local. 


            git clone

Cree un archivo HTML y agregue algo de contenido.


            <!DOCTYPE html>
            <html>
            <head>
            <title>Mi sitio web</title>
            </head>
            <body>
            <h1>Hola, mundo!</h1>
            </body>
            </html>

Agregue el archivo al repositorio y realice un commit.


            git add index.html
            git commit -m "primer commit"

Finalmente, envíe los cambios al repositorio remoto.


            git push origin master
Configuración Github

Vaya a la configuración del repositorio en Github y desplácese hacia abajo hasta la sección de `Pages`.

En la sección de Github Pages, seleccione la rama principal como fuente y haga clic en Guardar.

Tienes que tener el repositorio público para que funcione o pagar para poder publicar en privado.

Supervisar el proceso

Una vez que se haya configurado Github Pages, puede supervisar el proceso de implementación en la pestaña `Actions` de su repositorio.

En la pestaña de `Actions`, puede ver el estado de la implementación y solucionar problemas si algo sale mal.

URL final

Después de que se complete la implementación, su sitio web estará disponible en la URL proporcionada en la sección de Github Pages de la configuración del repositorio.

Puede compartir esta URL con otros para que puedan ver su sitio web en línea.

Fuerza bruta

Si se quiere forzalo desde el terminal del editor de código, se tiene que instalar el paquete de Github Pages con nodeJS. 


              npm install gh-pages

Luego se tiene que agregar el script en el archivo package.json. 


              "scripts": {
                "deploy": "gh-pages -d build"
              }

Finalmente se tiene que ejecutar el script.


              npm run deploy
